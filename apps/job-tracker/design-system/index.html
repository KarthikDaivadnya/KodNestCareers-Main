Build a job application tracker table with the following columns:
- Company Name (text)
- Role Title (text)
- Date Applied (date)
- Status (dropdown: Applied / Screening / Interview / Offer / Rejected)
- Notes (text, optional)

Requirements:
- Clean table with sticky header
- Each row has a status badge (colour-coded)
- Clicking a row opens a detail panel on the right
- Empty state shows when no applications exist
- Mobile: collapse to card list view
